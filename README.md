# 🧠 Project Overview

This project implements a **Retrieval Augmented Generation (RAG)** system that answers **Data Structures & Algorithms questions** using a custom PDF knowledge base.

Instead of relying purely on LLM knowledge, the system:

✔ Retrieves relevant information from documents  
✔ Uses **semantic vector search**  
✔ Generates **accurate AI responses using Gemini**

This architecture helps **reduce hallucination and improve reliability** of AI answers.

# ⚡ Features

🚀 Retrieval Augmented Generation (RAG)  
📚 Document based knowledge system  
🔎 Semantic vector similarity search  
💬 Conversational AI with query rewriting  
📄 PDF ingestion pipeline  
⚡ Fast vector search using Pinecone

# 🧰 Tech Stack

| Technology | Purpose |
|------------|--------|
| **Node.js** | Backend runtime |
| **LangChain** | LLM orchestration framework |
| **Google Gemini API** | AI model for answer generation |
| **Pinecone** | Vector database for semantic search |
| **Embeddings Model** | Convert text to vectors |
| **PDF Loader** | Document ingestion |
